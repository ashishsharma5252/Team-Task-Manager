# Team Task Manager

A full-stack MERN application for managing team tasks and projects.

## Features

- User Authentication
- Task Management
- Project Management
- Dashboard Analytics
- Role-based Access

## Tech Stack

### Frontend
- React
- Vite
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Installation

### Clone Repository

```bash
git clone https://github.com/ashishsharma5252/Team-Task-Manager.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create `.env` file inside backend:

```env
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
PORT=your_port_number
```

## Deployment

- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
